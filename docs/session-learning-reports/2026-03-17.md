## Learning Candidates

### Candidate 1: GitHub Actions secrets in `if` expressions invalidate workflows

- **Score:** 4 (repeated pattern +3, DX friction +1)
- **Source:**
  [zap-baseline.yml](/C:/dev/Updog_restore/.github/workflows/zap-baseline.yml),
  GitHub Actions runs `23224844394`, `23223711713`, `23213864085`
- **Category:** Infrastructure / CI
- **Anti-Pattern:** Referencing `secrets.*` directly inside workflow or job
  `if:` expressions makes the run fail before any jobs start. The symptom is a
  GitHub Actions run with `0` jobs, no failed-job logs, and the generic message
  "This run likely failed because of a workflow file issue."
- **Fix:** Move the secret into `env` or a setup step, then gate later steps
  with `env.*` or step outputs. In this case, `ZAP_BASE_URL` was copied into job
  `env`, a skip step was added for the unset case, and the ZAP action step now
  runs only when `env.ZAP_BASE_URL != ''`.
- **Recommendation:** CREATE REFLECTION (REFL-029)

### Candidate 2: Import-time config evaluation leaks environment requirements into unit tests

- **Score:** 5 (repeated pattern +3, CI breakage +1, DX friction +1)
- **Source:** [jwt.ts](/C:/dev/Updog_restore/server/lib/auth/jwt.ts),
  [requireFundAccess.test.ts](/C:/dev/Updog_restore/tests/unit/auth/requireFundAccess.test.ts),
  GitHub Actions run `23224844728`
- **Category:** Infrastructure / Testing
- **Anti-Pattern:** A module that exposes both pure request helpers and
  config-dependent auth helpers performs `getConfig()` at import time. Tests
  that only use the pure helper still fail in CI when unrelated env like
  `DATABASE_URL` is empty, because config validation executes before the test
  path is reached.
- **Fix:** Keep config access lazy. `server/lib/auth/jwt.ts` now resolves config
  only inside JWT verification, JWKS initialization, and token signing paths, so
  `requireFundAccess` can be imported without triggering `DATABASE_URL`
  validation.
- **Related:** REFL-024
- **Recommendation:** UPDATE EXISTING REFL-024 with this auth/config case
