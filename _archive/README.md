# Archive Directory

This directory contains historical files, logs, temporary scripts, and other
artifacts moved from the root directory to reduce clutter.

## Structure

- **docs-2025/**: Historical documentation, handoff memos, phase plans, and
  summaries from 2025.
- **logs/**: Execution logs, error reports, performance profiles, and large JSON
  history files.
- **scripts/**: One-off maintenance scripts, fix scripts, and obsolete
  automation scripts.
- **tmp/**: Temporary files (e.g., `tmpclaude-*`) and other transient data.
- **patches/**: Git patch files.
- **[Other Directories]**: Archived backups and test reports (e.g., `.backup`,
  `artifacts`, `playwright-report`).

## Restoration

If you need to restore any file, simply move it back to its original location.
Check the file dates to determine relevance.
